## 📖 About CalcMaster

**CalcMaster** is a modern Android calculator application designed to make everyday calculations simple and fast.

It includes calculation history, voice input, currency conversion, unit conversion, multiple languages, themes, a math game and a home-screen widget.

---

## ✨ Main Features

### 🎨 Beautiful User Interface

Clean, modern and user-friendly design for a smooth experience.

### 🧮 Basic Calculator

Perform simple and advanced calculations quickly and easily.

### 🕘 Calculation History

Automatically saves previous calculations so users can view them later.

### 🎤 Voice Input

Speak a calculation instead of typing it manually.

### 🔊 Voice Result

The app can speak the calculated result.

> “Your answer is 20.”

Voice output can be turned **ON or OFF**.

### 🏠 Home-Screen Widget

Quickly access the calculator directly from the device home screen.

## 💱 Currency Converter

CalcMaster uses the **ExchangeRate API** to provide currency conversion.

Users can:

- Enter an amount
- Select source and destination currencies
- Convert the entered amount
- View country and currency names
- Receive updated exchange-rate results

> [!IMPORTANT]
> Do not upload your private API key directly to GitHub.  
> Store it securely and exclude it using `.gitignore`.
